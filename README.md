# Quokka Adventures

A joyful website for Quokka Adventures tours on Rottnest Island, Western Australia.

## Website Structure

- **index.html** - Main website with services, team, FAQ, and contact information
- **styleguide-index.html** - Brand style guide with typography, colors, and brand assets
- **style.css** - Main website styles
- **styleguide-style.css** - Style guide specific styles

## Deployment

This site is deployed using GitHub Pages. The deployment is automated via GitHub Actions.

### Accessing the Style Guide

The brand style guide is accessible from the main navigation menu or directly at `/styleguide-index.html`.

## Local Development

To run the site locally:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000` in your browser.
