# Quokka Adventures

A joyful, authentic website for Quokka Adventure tours on Rottnest Island, Western Australia.

## Files

- `index.html` - Main website homepage
- `styleguide-index.html` - Brand style guide with typography, colors, and brand voice
- `style.css` - Main stylesheet
- `styleguide-style.css` - Style guide specific styles
- `images/` - Image assets including logos and photos

## Deployment

This website is automatically deployed to GitHub Pages when changes are pushed to the `main` branch.

### GitHub Pages Setup

The site uses GitHub Actions for automated deployment. The workflow is defined in `.github/workflows/deploy.yml`.

### Accessing the Deployed Site

Once deployed, the site will be available at:
- Main site: `https://thedalycreative.github.io/eqc-quokka-adventures/`
- Style guide: `https://thedalycreative.github.io/eqc-quokka-adventures/styleguide-index.html`

### Manual Deployment

To manually trigger a deployment:
1. Go to the Actions tab in the GitHub repository
2. Select the "Deploy to GitHub Pages" workflow
3. Click "Run workflow"

## Local Development

To view the site locally:
1. Clone the repository
2. Open `index.html` or `styleguide-index.html` in a web browser

No build process is required as this is a static HTML site.
